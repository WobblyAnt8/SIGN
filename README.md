<h2>Вариант №17.</h2>
<h3>Задание: </h3> 
<p>Определить класс с именем SIGN, содержащий следующие поля:</p>
<ul>
<li>Фамилия, имя;</li>
<li>Знак зодиака;</li>
<li>День рождения(массив из трех чисел)</li>
</ul>
<p>Определить методы доступа к этим полям и перегруженные операции извлечения и вставки для объектов типа РRICE.</p>
<p>Написать программу, выполняющую следующие действия:</p>
<ul>
<li>ввод с клавиатуры данных в массив, состоящий из восьми объектов типа SIGN; записи должны быть упорядочены по знакам зодиака; </li>
<li>вывод на экран информации о людях, родившихся в месяце, значение которого введено с клавиатуры;</li>
<li>если таких нет, выдать на дисплей соответствующее сообщение.</li>
</ul>
